# reminder_bot
It is a discord bot that creates reminders by giving the following format:

 - RAW: ```!command dd/mm/yyyy hh:mm message-like-this```
 - EXAMPLE: ```!reminder 19/Mar/2023 03:28 test-for-nuggets```
